# CryLens iOS

This directory will contain the SwiftUI iOS frontend for CryLens.

## Requirements

- macOS (Ventura or later recommended)
- Xcode 15+
- Apple Developer account (for device testing / distribution)

iOS development requires Xcode, which is only available on macOS. If you are on Windows or Linux, you can still develop and test the full backend — see [../backend/README.md](../backend/README.md).

## Planned features

- Record baby cry audio and send to backend for classification
- View per-baby cry history and label breakdown charts
- Push notification alerts for detected cry patterns
- Multi-baby support with separate profiles
